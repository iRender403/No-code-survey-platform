## *复盘：* 初识 AntDesign 中的 Design Token
全局token:所有的容器边框，都会收到影响
组件token:只设置对应的组件
- 如何修改不清楚的内容
1. 查文档，Design Token里面查名字
2. 全局改写在theme.token里面
3. 局部修改，放在component里面
4. 对于局部修改的部分应该再次嵌套一个configprovider

Token 与 CSS 变量的关系
1. 变量化 ： 当定义了一个Token 底层引擎就会进动态生成CSS变量
2. 消费变量 ：组件内部引用这些变量
3. 实时计算

之所以要使用token,其实其本质上是在修改底层的CSS变量，从而实现全局或局部样式实时同步与主题切换
